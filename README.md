## 頁面說明

1. **首頁/清單頁 (index.html)**

   - 搜尋欄
   - 分類選單
   - 動作清單(含縮圖、名稱、難度和效果標籤)
   - 語言切換按鈕

2. **詳細頁 (detail.html)**

   - 動作詳細資訊(名稱、難度、效果、注意事項)
   - 示範圖片(可點擊放大)
   - 教學影片
   - 語言切換按鈕

3. **關於頁 (about.html)**

   - 應用程式介紹
   - 開發背景
   - 開發者資訊
   - 語言切換按鈕

4. **聯絡頁 (contact.html)**
   - 聯絡表單
   - 聯絡資訊
   - 常見問題
   - 語言切換按鈕

## 如何使用

1. 直接在瀏覽器中開啟 `index.html` 檔案即可使用本應用
2. 無需安裝任何額外套件或伺服器環境
3. 點擊頂部的地球圖標可切換語言
